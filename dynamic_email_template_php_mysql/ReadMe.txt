

============ Instruction ============
1. Create a database called "codexworld" at phpMyAdmin and import the "email_templates.sql" file into this database.
2. Open the "dbConfig.php" file and modify the $dbHost, $dbUsername, $dbPassword, and $dbName variable's value with your database credentials.
3. Open the "add-template.php" file on the browser and create a new email template.
4. Open the "sendEmail.php" file on the editor ==> Specify the type of the email template and specify the user details source.
5. Now run the "sendEmail.php" script on the browser and test the email template functionality.
